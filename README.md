Creative Cyberpunk Portfolio

A high-performance, responsive personal portfolio website featuring a futuristic cyberpunk aesthetic. This project showcases technical skills, educational background, and creative projects through an immersive, interactive user interface.

Features
Cyberpunk Aesthetic: Immersive dark theme with neon accents, glowing borders, and glassmorphism effects.

Interactive UI Elements:

Custom Cursor: A dynamic gradient cursor with pulse animations and hover effects.

Animated Background: A multi-layered gradient background with a radial pulse animation.

Tab System: Smoothly transition between different sections (Home, About, Skills, Education, Projects, Contact) using custom animations.

Visual Effects:

Glitched Typography: Animated "glitch" effect on primary headings.

Interactive Project Cards: Hover-triggered scaling, 3D shadows, and neon shimmer effects.

Particle System: Floating background particles with randomized colors and animation durations.

Skill Bars: Animated progress bars featuring a shimmering light effect.

Responsive Design: Fully optimized for mobile, tablet, and desktop viewing.

Built With
HTML5: Semantic structure for web content.

CSS3 / Tailwind CSS: Custom styling including advanced animations (@keyframes), flexbox/grid layouts, and utility-first design.

JavaScript: Logic for the custom cursor, tab switching, particle generation, and server-side project loading.

Google Fonts: Utilizes specialized fonts like Orbitron, Rajdhani, Space Grotesk, and JetBrains Mono for a technical feel.

Project Structure
The portfolio is organized into several key sections:

Home: Introduction and primary call-to-action.

About: Personal professional summary.

Skills: Technical proficiency visualized through animated bars.

Education: A styled timeline of academic achievements.

Projects: Interactive display of bootcamp and server-hosted projects.

Contact: Direct communication links and contact form.

⚙️ Development Notes
Server Functionality: The site includes built-in scripts to check server status every 30 seconds and dynamically load projects from a backend.

Local Storage: Uses localStorage to save and persist specific user states or project data.
